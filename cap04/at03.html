1. Internet

A internet conecta computadores do mundo inteiro.

Os sites usam:

  1. HTML → estrutura

2. HTML

HTML significa:

HyperText Markup Language

O HTML cria a estrutura do site.

Estrutura básica 

<!DOCTYPE html>

<html>

<head>

  <title>Meu site</title>

</head>

<body>

  Conteúdo

</body>

</html>

Principais tags HTML

Tag Função
html Estrutura principal da página
head Configurações do site
body Conteúdo visível
title Nome da aba
meta Configurações extras

13. JavaScript

JavaScript adiciona lógica e interatividade.

Alert 

alert("Olá");

Variáveis 

let nome = "Erick";

Variáveis guardam dados.

Funções 

function falar(){

  alert("Oi");

}

Console 

console.log("Teste");

Pegar elemento HTML 

document.getElementById("titulo");

Alterar texto HTML 

document.getElementById("titulo")
.innerHTML = "Novo texto";

Evento onclick 

onclick="falar()"

Condição 

if(idade >= 18){

  alert("Maior");

}

Repetição 

for(let i = 0; i < 5; i++){

  console.log(i);

}
