Resumo: HTML, CSS e JavaScript

HTML = estrutura

CSS = aparência

JavaScript = lógica

HTML

HTML cria a estrutura do site.

<h1>Título</h1>
<p>Parágrafo</p>
<a href="">Link</a>
<img src="foto.png">
  • h1 até h6 = títulos
  • p = parágrafo
  • a = link
  • img = imagem
  • br = quebra de linha
  • hr = linha horizontal

</ul>
